## Description

BoomTown is a platform for sharing and borrowing items .

### Technologies used

```bash

Server: nodejs  expressjs Apollo GraphQL Postgres
Client: React  Redux  Apollo React+Router
Styling: Material-UI

```

### Installation

```bash
npm install
```

### Run Server

```bash
npm run start:dev
```

### While in the development environment, to start the project

```bash
npm start
```
